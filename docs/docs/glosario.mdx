# Glosario

- **Express Architecture**: Metodología visual e incremental para generar artefactos de decisión a partir de información mínima suficiente.
- **Metodología visual incremental**: Enfoque que prioriza representaciones visuales y mejoras iterativas.
- **Artefacto metodológico**: Resultado estructurado que habilita decisiones (p. ej., canvas, matriz, dashboard).
- **Artefacto de decisión**: Un artefacto útil que permite tomar o justificar una decisión organizacional.
- **Información Mínima Suficiente (IMS)**: Conjunto mínimo de variables necesarias para una decisión.
- **Trazabilidad organizacional**: Conexión explícita entre procesos, sistemas, métricas y decisiones.
- **Variable organizacional**: Elemento de información (proceso, sistema, métrica, responsable, etc.).
- **Proceso**: Secuencia de actividades que transforma entradas en salidas.
- **Capacidad**: Capacidad operativa o técnica para realizar actividades.
- **Sistema de información**: Aplicación o conjunto de herramientas que soportan procesos.
- **Herramienta**: Software o recurso que ayuda en la ejecución de actividades.
- **Dependencia**: Relación de dependencia entre elementos (sistemas, procesos, áreas).
- **Restricción**: Limite o condición que afecta soluciones posibles.
- **Métrica**: Indicador cuantitativo o cualitativo usado para medir desempeño.
- **Decisión habilitada**: Decisión que queda posible gracias a la información y artefactos generados.
- **Victoria temprana de gestión**: Mejora inmediata y tangible obtenida tras aplicar la metodología.
- **Design Science Research (DSR)**: Enfoque de investigación que diseña y evalúa artefactos para resolver problemas del mundo real.
- **Mini ADR**: Documento breve que registra una decisión técnica o de diseño con contexto y alternativas.
