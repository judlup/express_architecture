# Catálogo de Variables

Lista de variables organizacionales que se usan habitualmente:

- Área
- Proceso
- Actividad
- Actor
- Responsable
- Sistema
- Herramienta
- Entrada
- Salida
- Métrica
- Tiempo
- Capacidad
- Restricción
- Dependencia
- Riesgo
- Decisión

Cada variable debe documentarse con la mínima información suficiente para la decisión objetivo.

## Plantilla de documentación de una variable

- **Nombre:** (ej. Proceso)
- **Descripción breve:** qué representa la variable.
- **Nivel:** organización / área / proceso / actividad.
- **Responsable:** rol o persona.
- **Sistemas asociados:** lista de sistemas relevantes.
- **Métrica asociada:** indicador y frecuencia (si aplica).
- **Dependencias clave:** otras variables o procesos que afectan.
- **Decisión soportada:** ejemplo de decisión que habilita.

Mantener este catálogo actualizado facilita construir artefactos consistentes y posibilita trazabilidad entre elementos.
