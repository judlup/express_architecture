# Dashboard Operativo

## Propósito

Presentar métricas relevantes para la operación diaria y la toma de decisiones rápidas.

## Cuándo usarlo

- Monitorización de operaciones y seguimiento de SLAs.
- Identificación de tendencias que requieren intervención.

## Qué captura

- Indicadores clave (KPI), tendencias y alertas.
- Responsables por indicador y frecuencia de actualización.
- Umbrales y acciones recomendadas.

## Cómo construirlo

1. Seleccionar KPIs relevantes alineados con la decisión objetivo.
2. Definir frecuencias y responsables.
3. Visualizar con gráficos simples y alertas claras.

## Errores comunes

- Sobreexposición de métricas irrelevantes.
- Falta de propietarios para indicadores.
