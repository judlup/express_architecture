# Scorecard / Tablero Ejecutivo

## Propósito

Ofrecer una vista condensada para la dirección con indicadores estratégicos y decisiones prioritarias.

## Cuándo usarlo

- Seguimiento ejecutivo mensual / trimestral.
- Presentaciones a sponsors o juntas directivas.

## Qué captura

- KPI estratégicos y su tendencia.
- Estado de iniciativas clave y riesgos.
- Acciones prioritarias y responsables.

## Cómo construirlo

1. Seleccionar 6–10 indicadores relevantes al objetivo estratégico.
2. Resumir estado de iniciativas y riesgos asociados.
3. Indicar decisiones requeridas y propietarios.

## Errores comunes

- Incluir demasiados KPI que diluyen la atención.
- No vincular KPI a acciones concretas.
