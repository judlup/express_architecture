# Glosario

- **Express Architecture**: Metodología ligera y visual para apoyar decisiones.
- **Metodología**: Conjunto de conceptos, reglas y prácticas que estructuran artefactos y su uso.
- **Información Mínima Suficiente (IMS)**: La mínima información necesaria para una decisión.
- **Artefacto**: Elemento documentado (canvas, diagrama, matriz) que apoya decisiones.
- **ADR**: Architecture Decision Record.
- **DSR**: Design Science Research.
