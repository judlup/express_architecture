# Glosario

- **Express Architecture**: Metamodelo ligero y visual para apoyar decisiones.
- **Metamodelo**: Conjunto de conceptos y reglas que estructuran artefactos.
- **Información Mínima Suficiente (IMS)**: La mínima información necesaria para una decisión.
- **Artefacto**: Elemento documentado (canvas, diagrama, matriz) que apoya decisiones.
- **ADR**: Architecture Decision Record.
- **DSR**: Design Science Research.
