# Flujo general

El flujo general de trabajo en Express Architecture es práctico y orientado a decisiones. Pasos principales:

1. Contexto y decisión: definir claramente qué se quiere comprender o decidir.
2. Captura estructurada: recolectar información relevante mediante entrevistas, talleres y análisis de artefactos existentes.
3. Filtrado IMS: aplicar el criterio de Información Mínima Suficiente para quedarnos con lo esencial.
4. Diseño estructural: elegir el artefacto visual adecuado según la decisión.
5. Levantamiento y construcción: mapear variables y construir el artefacto (canvas, matriz, mapa, dashboard).
6. Generación visual: producir la representación visual clara y validable.
7. Validación: revisar con stakeholders, ajustar y acordar acciones.
8. Identificación de victorias tempranas: documentar mejoras rápidas y acciones priorizadas.
9. Evaluación y refinamiento: medir utilidad y ajustar la metodología.

El flujo es iterativo: cada ciclo busca producir artefactos accionables y aprendizajes que alimenten la siguiente iteración.
