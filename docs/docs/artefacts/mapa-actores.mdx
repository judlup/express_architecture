# Mapa de Actores

Propósito: mostrar quién participa, quién decide, quién ejecuta, quién provee información y quién se ve afectado.

Cuándo usarlo: al iniciar el levantamiento de procesos o al preparar una decisión que requiere múltiples actores.

Qué captura: actores, roles, responsabilidades, canales de comunicación y nivel de influencia.
