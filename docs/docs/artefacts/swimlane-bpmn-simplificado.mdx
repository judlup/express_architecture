# Swimlane / BPMN simplificado

Uso: entender flujos y responsabilidades entre áreas. Evitar detalle técnico; centrarse en pasos relevantes para la decisión.

Qué captura: actividades, responsables por carril, entradas, salidas y puntos de transferencia.
