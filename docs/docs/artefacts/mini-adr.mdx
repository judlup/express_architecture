# Mini ADR

Estructura mínima:

- Contexto
- Decisión
- Alternativas consideradas
- Consecuencias
- Responsables
- Fecha
- Evidencias usadas

Uso: documentar decisiones que afectan diseño, integraciones o prioridades.
