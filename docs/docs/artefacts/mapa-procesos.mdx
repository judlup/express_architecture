Propósito: mostrar el flujo de valor organizacional, entradas, salidas y responsables.

Qué captura: procesos, subprocesos, responsables, entradas, salidas y puntos de decisión.

Ejemplo de uso: diagnóstico inicial para identificar cuellos de botella y dependencias.
