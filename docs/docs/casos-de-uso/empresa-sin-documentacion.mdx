# Empresa sin documentación formal

Contexto: organización que depende de conocimiento tácito.

Problema: falta de trazabilidad y riesgo de pérdida de conocimiento.

Decisión a apoyar: priorizar procesos para documentación mínima suficiente y asignar responsables.

Artefactos: mapa de actores, mapa de procesos, matriz proceso-herramienta, mini ADR.
